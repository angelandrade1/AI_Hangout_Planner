## Getting Started

### 1. Setup Virtual Environment

From the API root:

```bash
make setup-venv
```

### 2. Install Dependencies

From the APIS root:

```bash
make install
```

### 2. Configure Environment Variables

**Mac OS**
```bash
make init-env-mac
```

**Windows OS**
```bash
make init-env-windows
```

### 3. Start the API Server

From the `API/` directory, run:

**Localhost Server**
```bash
make run-local
```
